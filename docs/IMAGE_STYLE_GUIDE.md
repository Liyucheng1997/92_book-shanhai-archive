# 《山海异志》统一图像风格规范

## 1. 风格定位

项目中的异兽图像统一采用“清末彩绘异兽谱 × 工笔重彩 × 传统连环画”的视觉体系。参考特征是高密度墨线、墨绿描金、米黄旧纸、山水云气和古籍卡页式装饰。画面应当像一页民间流传的精绘异兽册：叙事直观、装饰浓郁、轮廓有力，同时保留可信的身体结构和明确的物种辨识度。

核心气质：苍古、奇诡、华丽、民间异兽谱、可考据。  
核心介质：旧宣纸、矿物重彩、密集墨线、墨绿描金、局部朱砂。  
统一色盘：墨黑 `#25241D`、墨绿 `#193A34`、赭石 `#8A5039`、朱砂 `#8B3328`、旧金 `#AA8650`、纸黄 `#DEC9A2`、石青 `#31574D`。

## 2. 固定构图

- 比例：竖版 4:5，建议 2048 × 2560；主体完整，不裁断头、角、尾或四肢。
- 视角：略低于主体眼睛的三分之二侧面，全身像；主体占画面高度约 72%。
- 光源：古画式均匀散射光，主体只用墨线和色块塑造体积，禁止摄影棚高光与霓虹光。
- 背景：完整的古典山水叙事场景，以云气、山石、松树、水瀑表现栖息地；细节浓度约为主体的 50%。
- 边缘：主体外轮廓使用有粗细变化的强墨线，内部鳞片、毛发以密集细线刻画，整体带木版套色与手绘重彩质感。
- 装饰：允许回纹、如意云头、墨绿描金边框；图像内部仍禁止生成文字，标题由前端叠加，确保准确。
- 禁止在图内生成文字、印章、边框、图鉴 UI；这些内容由前端统一叠加。

## 3. 生物设计规则

1. 首先忠于原文形貌；原文没有描述的部位，以真实动物解剖结构补足，并在元数据中标为“艺术推定”。
2. 混合生物必须呈现自然的骨骼与肌肉过渡，不能像多种动物部件直接拼接。
3. 神能通过环境变化表达，不使用法阵、粒子特效、光剑或游戏技能特效。
4. 眼睛、鳞片、毛发等微观细节清晰，但不追求油亮的 3D 渲染质感。
5. 每只异兽只设置一个视觉记忆点；烛龙为“开合双目对应昼夜”。

```text
形貌约束：【只写原文明示的形貌】。

环境：【古籍所载地点及少量地貌】。
神能表现：【通过天色、气候、植被等环境表现，不使用魔法特效】。

视觉风格：清末彩绘异兽谱、工笔重彩与传统连环画结合，旧宣纸底，高密度墨线，墨绿描金，浓郁但陈旧的矿物色，强轮廓和细密内部刻画，完整古典山水场景，回纹与如意云头装饰，像一页精绘民间异兽册。竖版 4:5，全身三分之二侧面，主体占画面高度约 72%。

不要文字，不要印章，不要边框，不要 UI，不要现代物件，不要动漫，不要游戏卡牌，不要西方奇幻龙，不要霓虹光，不要塑料感 3D，不要对称摆拍，不要多余肢体，不要裁断主体。
```

## 5. 烛阴 / 烛龙专用提示词

```text
为《山海异志》数字博物图鉴绘制“烛阴（烛龙）”全身像。

古籍依据：“钟山之神，名曰烛阴。视为昼，瞑为夜；吹为冬，呼为夏。身长千里。在无䏿之东。其为物，人面，蛇身，赤色，居钟山下。”
形貌约束：人面，细长蛇身，通体赤色；是掌管昼夜寒暑的古老神祇，不画成有翼、有爪、有鹿角的常见东方龙。
艺术推定：面容为非特定族群的古老中性人面，与蛇颈自然衔接；鳞片参考赤链蛇与蚺蛇，身体具有真实重量感。
环境：钟山之下，北方荒寒山地。画面左侧为微明白昼，右侧为幽暗长夜；近处一侧草木结霜，另一侧有暑气与微红云霞。所有变化都来自它的睁眼、闭眼与呼吸，不出现法阵或粒子特效。

视觉风格：清末彩绘异兽谱、工笔重彩与传统连环画结合，旧宣纸底，高密度墨线，墨绿、暗金、赭石和朱砂的陈旧矿物色，强轮廓与极细鳞片刻画。背景为云气缭绕的钟山、松树与山瀑，四角有墨绿描金回纹装饰，像一页精绘民间异兽册。竖版 4:5，全身三分之二侧面，巨大的蛇身盘绕但轮廓完整。

不要文字，不要印章，不要边框，不要 UI，不要动漫，不要游戏卡牌，不要西方恶龙，不要龙角，不要翅膀，不要爪，不要盔甲，不要霓虹，不要过度恐怖血腥，不要塑料感 3D，不要多余头部和肢体，不要裁断主体。
```

## 6. 生成后验收

- 与原文冲突的结构直接淘汰，不靠后期说明补救。
- 同批图像的纸张底色、主体比例、光向和色彩饱和度必须一致。
- 图像输出后另存 `webp`（主图，质量 85）与 `avif`，保留原始无损文件。
- 文件命名：`creature_<英文 slug>_v<版本号>.<扩展名>`，例如 `creature_zhulong_v01.webp`。
- 图像元数据另存同名 JSON，至少记录原文依据、艺术推定、提示词版本和生成日期。
