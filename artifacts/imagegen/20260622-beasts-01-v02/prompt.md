# 南山经兽类第一批 V02 · 统一 4:5

Input image 1 仅作画风参考；Input image 2 仅作烛龙标准 4:5 画布比例参考。

生成一张严格 2×2 母板。整张母板必须是与烛龙完全相同的精确竖版 4:5，不是 2:3、5:8 或其他更狭长比例。四格边界严格位于宽高 50%，每格同为 4:5。顺序：左上狌狌、右上鹿蜀、左下类、右下猼訑。

- 狌狌：棕色猿猴形，白耳，伏行与人走的奔跑姿态。
- 鹿蜀：马形，白首，虎纹，单一赤尾，四只马蹄。
- 类：狸形，有颈背长髦，中性体态，单尾四足。
- 猼訑：羊形，九条可数尾巴、四只可数耳朵、双眼在背，脸上无眼。

统一采用项目既定的清末彩绘异兽谱、工笔重彩、旧宣纸、墨绿描金、传统山水与回纹边饰。每格只画一只完整全身异兽，主体和边框距离格边至少 8%，不得跨格。禁止文字、数字、印章、标签、水印、自由拼贴、现代物件、动漫、游戏卡牌、3D、霓虹、错误肢体和裁断主体。
